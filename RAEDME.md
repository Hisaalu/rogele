ROGELE (Rays of Grace E-Learning) is a comprehensive Learning Management System designed to facilitate online education with role-based access for admins, teachers, and students.

## Features

### Admin Features
- Full user management (students, teachers, admins)
- Class and subject management
- System analytics and reporting
- Content approval workflow
- Subscription and trial management

### Teacher Features
- Create and manage lessons with rich content
- Create quizzes with multiple question types
- Track student progress and performance
- Grade assignments and quizzes
- View class analytics
- Download student results (CSV/Excel)

### Student Features
- Access lessons and course materials
- Take quizzes and track scores
- View progress reports
- Bookmark important content
- Track completion rates
- View performance analytics
